### 3. Configure NetFlow on Cisco

~~~
!@CSR1000v
conf t
 no flow record FLOW_RECORD
 flow record FLOW_RECORD                   !- Collect Specific Packet Info
  match ipv4 source address
  match ipv4 destination address
  match transport source-port
  match transport destination-port
  match ipv4 protocol
  match ipv4 version
  match interface input
  match interface output
  collect counter bytes
  collect counter packets
  collect transport tcp flags
  collect flow direction
  collect timestamp absolute first
  collect timestamp absolute last
  !
  collect ipv4 tos
  collect ipv4 ttl
  collect routing next-hop address ipv4
  collect application name
  collect routing source as
  collect routing destination as
 !
 no flow exporter FLOW_EXPORTER
 flow exporter FLOW_EXPORTER               !- Specify Where to send Flows (IR-ElasticSys via VMNet2)
  destination 192.168.102.133
  transport udp 2055
  template data timeout 60                 !- Required
  option interface-table timeout 60        !- Required
  option exporter-stats timeout 60         !- Required
 !
 no flow monitor FLOW_MONITOR
 flow monitor FLOW_MONITOR                 !- Combine EXPORTER & RECORD Info
  exporter FLOW_EXPORTER
  record FLOW_RECORD
 !
 interface GigabitEthernet1
  ip flow monitor FLOW_MONITOR input       !- CSR will Monitor Packets Going through VMNet1 (NAT)
  ip flow monitor FLOW_MONITOR output
  end
~~~
