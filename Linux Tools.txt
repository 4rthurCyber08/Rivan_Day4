FTP from Windows to Linux

Step 1: Activate FTP on windows

Step 2: Place Files to be transfered on the FTP directory

Step 3: From Linux (NetOps), use the following command:

- Go to the directory where the files must be placed

@netops
cd /etc/ansible

- Activate FTP on Linux

@netops

 > Name: Administrator
 > Pass: C1sc0123

- Check the contents of the FTP directory
@netops
